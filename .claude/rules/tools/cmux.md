# cmux Rules

Purpose: cmux キーボードショートカットリファレンス。
Source: <https://www.cmux.dev/docs/keyboard-shortcuts>

## Workspaces

| ショートカット | アクション               |
| -------------- | ------------------------ |
| ⌘+N            | 新規ワークスペース       |
| ⌘+1〜8         | ワークスペース直接移動   |
| ⌘+9            | 最後のワークスペースへ   |
| ⌘+⇧+W          | ワークスペースを閉じる   |
| ⌘+⇧+R          | ワークスペースをリネーム |

## Surfaces（タブ）

| ショートカット  | アクション         |
| --------------- | ------------------ |
| ⌘+T             | 新規サーフェス     |
| ⌘+⇧+[ / ⌃+⇧+Tab | 前のサーフェス     |
| ⌃+1〜8          | サーフェス直接移動 |
| ⌃+9             | 最後のサーフェスへ |
| ⌘+W             | サーフェスを閉じる |

## Split Panes

| ショートカット | アクション             |
| -------------- | ---------------------- |
| ⌘+D            | 右に分割               |
| ⌘+⇧+D          | 下に分割               |
| ⌥+⌘+←/→/↑/↓    | ペイン移動（方向指定） |
| ⌥+⌘+D          | ブラウザを右に分割     |
| ⌥+⌘+⇧+D        | ブラウザを下に分割     |

## Browser

| ショートカット | アクション               |
| -------------- | ------------------------ |
| ⌘+⇧+L          | ブラウザサーフェスを開く |
| ⌘+L            | アドレスバーにフォーカス |
| ⌘+]            | 進む                     |
| ⌘+R            | ページをリロード         |
| ⌥+⌘+I          | DevTools を開く          |

## Notifications

| ショートカット | アクション               |
| -------------- | ------------------------ |
| ⌘+⇧+I          | 通知パネルを表示         |
| ⌘+⇧+U          | 最新の未読通知へジャンプ |
| ⌘+⇧+L          | フラッシュをトリガー     |

## Find

| ショートカット | アクション        |
| -------------- | ----------------- |
| ⌘+F            | 検索              |
| ⌘+G / ⌘+⇧+G    | 次/前の一致へ移動 |
| ⌘+⇧+F          | 検索バーを閉じる  |
| ⌘+E            | 選択範囲で検索    |

## Terminal

| ショートカット | アクション               |
| -------------- | ------------------------ |
| ⌘+K            | スクロールバックを消去   |
| ⌘+C            | コピー                   |
| ⌘+V            | ペースト                 |
| ⌘++ / ⌘+−      | フォントサイズ 拡大/縮小 |
| ⌘+0            | フォントサイズをリセット |

## Window

| ショートカット | アクション     |
| -------------- | -------------- |
| ⌘+⇧+N          | 新規ウィンドウ |
| ⌘+,            | 設定           |
| ⌘+⇧+R          | 設定をリロード |
| ⌘+Q            | 終了           |

## Ghostty との連携

cmux のペイン移動は `⌥+⌘+←/→/↑/↓`。
Karabiner-Elements で `C-x h/j/k/l` → `⌥+⌘+←/↓/↑/→` にマッピングしている。
詳細: `.claude/rules/tools/karabiner.md`

> **注意**: `⌥+⌘+←/→` は Rectangle 等のウィンドウマネージャーと競合する場合がある。
> その場合は Rectangle 側の該当ショートカットを無効化すること。
